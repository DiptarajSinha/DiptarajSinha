An experimental, immersive portfolio designed to provide three distinct user experiences based on the visitor's intent.

---

## 🎭 The Three Paths

### 1. 📂 The Recruiter (Professional)

A streamlined, high-performance interface for career professionals. Focuses on **Experience**, **Skills**, and **Certifications** with a clean, high-contrast UI.

### 2. 🗺️ The Adventurer (Creative)

An "Explorer's Journal" aesthetic. Projects are treated as **Artifacts** found in the digital wilds.

- **Engagement:** Lens-focus loading animations, "Discovery Log" metadata, and 3D journal-opening transitions.

### 3. 👁️ The Stalker (Cyber-Goth / Hacker)

A dark, terminal-heavy simulation for those who like to "dig deeper."

- **Engagement:** Payload injection simulations, SSL pinning bypass logs, and strict access-denial protocols for restricted data nodes.

---

## 🛠️ Core Tech Stack

- **Frontend:** Next.js 14 (App Router)
- **Animations:** Framer Motion (Complex layout transitions & 3D transforms)
- **Icons:** React Icons (Fa/Io)
- **State Management:** React Hooks (UseEffect/UseState) with LocalStorage persistence for "Access Clearance."
- **Typography:** Custom themed fonts (Stalker Mono & Adventurer Serif)

---

## 🚀 Key Features

- **Role-Persistent Navigation:** The UI remembers your selected role across sessions.
- **Dynamic Modals:**
  - **Stalker:** Brute-force timers and temporary session grants.
  - **Adventurer:** Telescope/Binocular focusing effects for "Artifact Deciphering."
- **Responsive Hardware Simulation:** Optimized for mobile touch targets while maintaining the "Control Panel" feel on desktop.
- **Interactive Arcade:** Mini-games and UI experiments built directly into the site.

   ```bash

   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Run the development server:**
   ```bash
   npm run dev
   ```
4. **Build for production:**

```bash
npm run build
```
